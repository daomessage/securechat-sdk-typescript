## Quick Start

```ts
import { SecureChatClient, newMnemonic } from '@daomessage_sdk/sdk'

const client = new SecureChatClient()

// 注册
const mnemonic = newMnemonic()
console.log('BACK THIS UP:', mnemonic)
const { aliasId } = await client.auth.registerAccount(mnemonic, 'Alice')
console.log('your alias:', aliasId)

// 响应式好友列表
const sub = client.contacts.observeFriends().subscribe((friends) => {
  console.log('friends:', friends)
})

// 加好友
const profile = await client.contacts.lookupUser('bobshandle')
await client.contacts.sendRequest(profile.alias_id)

// 清理订阅
sub.unsubscribe()
```

## API 速查

### Auth

```ts
await client.auth.registerAccount(mnemonic, nickname)
await client.auth.loginWithMnemonic(mnemonic)
await client.auth.restoreSession()          // 返回 {aliasId, nickname} 或 null
await client.auth.logout()
```

### Contacts

```ts
// 观察式
client.contacts.observeFriends()              // Observable<FriendProfile[]>
client.contacts.observeAccepted()             // 只要 accepted
client.contacts.observePending()              // 只要 pending + received
client.contacts.observePendingCount()         // badge 用

// 命令式
await client.contacts.lookupUser(aliasId)
await client.contacts.sendRequest(aliasId)
await client.contacts.accept(friendshipId)    // 乐观更新 + rollback
await client.contacts.reject(friendshipId)    // 乐观更新 + rollback
```

### Messages

```ts
client.messages.observeConversations()        // Observable<ConversationSummary[]>
client.messages.observeMessages(convId)       // Observable<StoredMessage[]>

await client.messages.send({ conversationId, toAliasId, text })
await client.messages.markAsRead(conversationId)
```

### Media

```ts
const id = await client.media.sendImage(convId, file)
const id = await client.media.sendFile(convId, file)
const id = await client.media.sendVoice(convId, blob, durationMs)

client.media.observeUpload(messageId).subscribe((p) => {
  console.log(p.phase, p.loaded, p.total)
  // phase: 'encrypting' | 'uploading' | 'done' | 'failed'
})
```

### Security(身份验证 / 安全码)

```ts
client.security.observeTrust(contactId).subscribe((state) => {
  // { status: 'unverified' } | { status: 'verified', verifiedAt, fingerprintSnapshot }
})

const code = await client.security.getSafetyCode(contactId, myPubKey, theirPubKey)
// code.displayCode → "AB12 CD34 ..."

const ok = await client.security.verifyCode(contactId, userInput, myKey, theirKey)
// 匹配 → 自动 markVerified,observeTrust 自动 emit verified
```

### Calls

```ts
client.calls?.observeState().subscribe((state) => { ... })
client.calls?.observeLocalStream().subscribe((s) => { video.srcObject = s })
client.calls?.observeRemoteStream().subscribe((s) => { ... })

await client.calls?.start(peerAlias, { audio: true, video: true })
await client.calls?.accept()
await client.calls?.hangup()
```

### Events(全局流)

```ts
client.events.network.subscribe(...)       // 'connected' | 'connecting' | 'disconnected'
client.events.sync.subscribe(...)          // { tag: 'idle' | 'syncing' | 'done', ... }
client.events.error.subscribe(...)         // SDKError | null (initial null)
client.events.message.subscribe(...)       // 跨会话全局消息流
```

## 核心概念

### 订阅是零成本

所有 `observe*()` 返回的是 `BehaviorSubject` 视图:

- **订阅立即收到当前值**(不是等下次变化)
- **多订阅者共享同一份内部数据**(不会触发多次 HTTP)
- **取消订阅不影响别人**

React 里最简单的集成:

```tsx
function useObservable<T>(obs: Observable<T>): T {
  const [v, setV] = useState(obs.value)
  useEffect(() => {
    const s = obs.subscribe(setV)
    return () => s.unsubscribe()
  }, [obs])
  return v
}

// 使用
const friends = useObservable(client.contacts.observeFriends())
```

### 乐观更新是默认行为

`accept(friendshipId)` 瞬间改本地状态 → 订阅者看到变化 → 后台调服务器 → 失败自动 rollback。

**你不写 rollback,SDK 帮你写了**。

### 请求去重是默认行为

10 个组件同时 `observeFriends()` → SDK 内部 Mutex 保证只发一次 `/api/v1/friends`。

## React 完整示例

```tsx
import { SecureChatClient } from '@daomessage_sdk/sdk'
import { useEffect, useState } from 'react'

const client = new SecureChatClient()

function FriendList() {
  const [friends, setFriends] = useState([])
  useEffect(() => {
    return client.contacts.observeFriends().subscribe(setFriends).unsubscribe
  }, [])
  return (
    <ul>
      {friends.map((f) => (
        <li key={f.alias_id}>
          {f.nickname} — {f.status}
          {f.status === 'pending' && f.direction === 'received' && (
            <button onClick={() => client.contacts.accept(f.friendship_id)}>
              Accept
            </button>
          )}
        </li>
      ))}
    </ul>
  )
}

function NetworkBanner() {
  const [state, setState] = useState('disconnected')
  useEffect(() => {
    return client.events.network.subscribe(setState).unsubscribe
  }, [])
  if (state === 'connected') return null
  return <div>⚠️ {state}</div>
}
```

## 从 0.3.0 升级

如果你用过 0.3.0(内测版本):

1. **删掉 `attachReactive(client)`**:
   ```diff
   - const reactive = attachReactive(client)
   - reactive.contacts.observeFriends()
   + client.contacts.observeFriends()
   ```

2. **改 `import`**:
   ```diff
   - import { attachReactive } from '@daomessage_sdk/sdk/reactive'
   + import { SecureChatClient } from '@daomessage_sdk/sdk'
   ```

3. **类名去 Reactive 前缀**:
   ```diff
   - import { ReactiveContactsModule } from '@daomessage_sdk/sdk'
   + import { ContactsModule } from '@daomessage_sdk/sdk'
   ```

## FAQ

### Q: 为什么没有 `syncFriends()` 命令式 API?

0.4.0 认为"查询即订阅" — 你几乎从不会"只查一次就走",所以 `observeFriends().value`(一次性快照)或 `observeFriends().subscribe(handler)`(订阅)就够。

如果你真的只要一次:`client.contacts.observeFriends().value` 给你当前值。

### Q: 我需要手动 unsubscribe 吗?

- **React**:是(`useEffect` cleanup)
- **Compose (Android, 未来版本)**:否(`collectAsStateWithLifecycle` 自动处理)
- **SwiftUI (未来版本)**:否(`.task` 自动 cancel)
- **原生 Node.js 脚本**:是

不 unsubscribe 会泄漏内存。

### Q: 和 rxjs 关系?

不依赖 rxjs。SDK 自带 `Observable<T>` ~3KB 实现,语义和 `BehaviorSubject` 一致。你可以用 rxjs `from(client.contacts.observeFriends())` 转成 rxjs Observable。
